# media-server
Docker-Compose and configuration files to run the Media-Server on Raspberry Pi

## Docker containers being used

1. Heimdall        - Dashboard
2. Plex            - Media player
3. Transmisison    - Torrent downloader client
4. Sonarr          - Searches for TV shows to download
5. Radarr          - Searches for movies to download
6. LazyLibrarian   - Searches for books to download
7. Ombi            - Allows users to request shows/movies to add to Plex/Sonarr/Radarr
8. Jackett         - Allows Sonarr/Radarr to search torrent trackers
